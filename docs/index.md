# 研究方向
我的研究方向：算法工程，搜索及优化。
我研究了图上的一些组合搜索问题，例如最大团问题、松弛团问题，Steiner树问题，反馈集问题，以及人工智能领域中一些新兴的相关问题。
此外，我还关注算法的复杂性分析。



具体内容的可参考我的[DBLP](https://dblp.org/pid/01/1901-16.html)或者[google scholar](https://scholar.google.com/citations?user=8MvNCXAAAAAJ) 页面。

# 算法工程小组
我指导了[UESTC算法工程小组](aegroup/aegroup.md)，组员由对算法和程序设计有强烈兴趣的学生组成，我们探索各种的优化问题及其高效的求解技巧。
小组每周都有学术活动，包括小组讨论，算法比赛，听大牛报告[seminar](https://tcsuestc.com/seminars/)。
我们也会指导学生发表顶级论文，团队诞生了包括成电杰出学生（电子科大最高荣誉），华为天才少年等学生。
